WP Category Thumbnail create a thumbnail widget

== Description ==

This plugin provides thumbnails for categories fetched from the latest page/post of the corresponding category. The post thumbnail are generated either from its featured image or from the first available image in the post.

<a href="http://www.nettantra.com">NetTantra</a>

== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory
2. Change the perimission to your plugin 'scripts/cache' directory  e.g.: `chmod 0777 scripts/cache/`
3. Activate the plugin

= Features =

* Show thumbnails for latest page/post of the corresponding category
